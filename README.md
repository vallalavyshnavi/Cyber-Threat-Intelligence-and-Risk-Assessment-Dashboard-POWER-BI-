Cyber Threat Intelligence & Risk Assessment Dashboard

 Project Overview

This project presents an end-to-end Cybersecurity Analytics Dashboard developed using Power BI to analyze global cyber threats, financial impacts, risk levels, geographic attack patterns, and threat intelligence insights.

The dashboard enables organizations to monitor cybersecurity incidents, identify high-risk attack patterns, evaluate financial losses, and gain actionable intelligence through interactive visualizations and KPIs.

 Objectives

- Analyze global cybersecurity incidents from 2015–2024
- Identify attack trends and threat patterns
- Evaluate financial impact across industries and countries
- Assess cybersecurity risks using custom DAX measures
- Provide executive-level cybersecurity insights through interactive dashboards

Tools & Technologies

- Power BI
- Power Query
- DAX (Data Analysis Expressions)
- Data Visualization
- Data Modeling
- Business Intelligence

 Key Business Insights

- Low-risk incidents account for approximately 39.5% of total incidents.
- DDoS and Phishing attacks generate significant financial losses.
- The IT industry experiences the highest cybersecurity impact.
- High-risk incidents represent over 30% of all recorded incidents.
- Financial losses vary significantly across industries and attack types.
